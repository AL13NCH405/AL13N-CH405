SiteBroker!

A cross-platform python based utility for information gathering and penetration automation!
Requirements:
* Python (2.7.*)
* Python pip
* Python module requests
* Python module colorama
* Python module dnspython
* Python module lxml
* Python module bs4

Installation:
Step 1:
git clone https://github.com/Anon-Exploiter/SiteBroker
Step 2:
cd SiteBroker
Step 3:
pip install -r requirements.txt
Step 4:
python SiteBroker.py

WPScan!

A Ruby based utility for exploiting & inflitrtating WordPress Websites!
Prerequisites:
* Ruby >= 2.3 (Recommend Latest)
* Curl >= 7.21 (Recommend Latest)
* RubyGems (Recommend Latest)

Installation:
Step 1:
git clone https://github.com/wpscanteam/wpscan
Step 2:
cd wpscan
Step 3:
bundle install && rake install
Step 4:
wpscan --url sitetotest.com --arguments
